## 安装部署教程

直接Clone本项目，首先先在数据库中运行[数据库文件](https://gitee.com/azeng233/smbms/blob/master/src/main/resources/smbms.sql)，然后配置好Tomcat以及Maven导包，就可以直接在本机完美运行此项目！👨
